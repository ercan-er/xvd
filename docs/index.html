xvd --features

Built for the command line.
No compromises.

Single download

Paste any X / Twitter video URL and get the highest quality MP4 instantly.

xvd https://x.com/user/status/…

Batch download

Feed a text file full of URLs. Up to 8 parallel downloads, progress per line.

xvd --batch urls.txt -c 8

Watch mode

Copy any X link to your clipboard — xvd auto-downloads it in the background.

xvd --watch -o ~/Videos

GIF conversion

Convert any video to a high-quality animated GIF with one flag.

xvd URL --gif

Watermark

Burn a PNG logo into the video at any corner with custom size and opacity.

xvd URL --watermark logo.png

Subtitles

Fetch caption tracks, translate to any language, and burn permanently into the video.

xvd URL --subtitle tr

Transcription

No captions on the video? Auto-transcribe via OpenAI Whisper and burn subtitles.

OPENAI_API_KEY=sk-… xvd URL --subtitle tr

Profile scraping

Download every video from a profile. Filter by date or keyword.

xvd --profile @NASA --from 2024-01-01

History

Browse everything you've downloaded with an interactive terminal UI.

xvd --history
